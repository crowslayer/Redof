# Redof
### Routine to download files from the DOF Yucatan, in bulk, directly or via scraping.

# Requirements.
- **request**
- **tqdm**
- **BeautifulSoup**

## Parameters

- **begin_date: Beginning date**
- **end_date: end date**
- **path: Directory path where the files will be downloaded.**
